## **Outcome**

* Used HTML5 **semantic elements** for accessability
* Used **Git** and **GitHub** for project management
* Tried to maintain **clean code**
* Used **media queries** for responsive design, implemented via **mobile first workflow**
* **Cross tested** on Firefox and Chromium based browsers

<br>

## **What I learned**

* Usage of `grid-template-area` css property to provide values for rows & columns itself without need of seperate declarations
* Utilization of `:active()` pseudo-class of `button` element, which acts as an alternative for `:hover` pseudo-class
* A lot of minor things
